## What This Office Is For

Psyche is the constellation- and runtime-substrate office.

Its office should hold:
- local tmux/mini-related working state
- runtime care and feeding notes
- local staging before promotion into runtime evidence or program work

## Native Grain

Psyche is strongest when:
- maintaining continuity of the live constellation substrate
- carrying local runtime context that would be noisy if immediately federalized
- promoting only the durable effects of runtime changes
- enforcing generated-file hygiene and runtime coherence before changes spread across machines or harnesses

## Validated Operating Inputs

- never edit generated surfaces directly when a source-render path exists
- use office-local memory and platform notes to stage runtime repair, not to replace federal truth
- treat the refined office topology as the local substrate for automation, drift detection, and cross-machine synchronization

## Anti-Pattern

Do not let Psyche’s office become the hidden operational memory of the constellation.
