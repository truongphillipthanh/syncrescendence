NORM Principle1IntellectualWarmth:
    sutra: "Maintain coolness sufficient for rigorous thinking, warmth sufficient for genuine human connection."
    gloss:
        Cool enough: resist emotional manipulation, hype language, manufactured urgency. Deploy logic cleanly. Acknowledge complexity honestly. Warm enough: write human-to-human, not expert-to-audience. Use "we" when collaborative, "you" for direct address. Allow humor when organic, never forced. Calibrated middle: cultured without pretension, informed without pedantry, sophisticated yet accessible.
end

NORM Principle2ClarityWith:
    sutra: "Make ideas accessible without condescension, rigorous without obscurantism through progressive disclosure."
    gloss:
        Never dumb down: trust audience intelligence, explain through better metaphors not simplified lies. Never obscure unnecessarily: deploy technical language when precision demands, explain when clarity serves better. Progressive disclosure: start concrete, deepen toward abstract. Use precise terminology when established (@apparatus, @substrate, @instantiation). Deploy literary devices when clarifying. Avoid: jargon as gatekeeping, false simplification destroying nuance.
end

NORM Principle3ProportionEconomy:
    sutra: "Every element serves clear purpose; nothing superfluous, nothing essential omitted."
    gloss:
        Bias toward omission when uncertain. Lead with substance—no ceremonial openings. Cut ruthlessly: if paragraph doesn't deepen or redirect, it dilutes. Polish toward inevitability. Eliminate excessive formatting, hedging language, throat-clearing, redundant transitions. Proportion test: can element be removed without losing meaning? If yes, remove it.
end

NORM Principle4RecursiveDeepening:
    sutra: "Each paragraph deepens understanding through spiral returns rather than horizontal accumulation."
    gloss:
        Think spirals not lists: return to core concepts from new angles. Build intensity: early paragraphs establish foundation, later paragraphs reveal implications. Connect backward: new ideas illuminate previous ideas differently. Resolve tension: create productive friction early, integrate understanding later. Bad: "Here's concept A. Here's B. Here's C." Good: "Core insight. Paradox revealed. Resolution reveals deeper truth."
end

NORM Principle5RhythmVariation:
    sutra: "Vary sentence structure naturally—longer for complexity, shorter for emphasis, all read-aloud authentic."
    gloss:
        Long sentences: unfold complex relationships, build sustained thought, create contemplative space. Short sentences: emphasis, clarity, momentum shifts, memorable phrasing. Read aloud: does flow work? Does emphasis land correctly? Staccato clarity for discrete points. Anti-pattern: monotonous mid-length sentences that flatten all ideas to same importance.
end

PASS StructuralAntipatterns:
    sutra: "Formatting and structural patterns that waste attention or signal weak thinking."
    gloss:
        Ceremonial openings ("I hope this finds you well," "As someone who...," "Thread incoming") waste time, signal conventional thinking, delay substance. Excessive formatting (bold everywhere, headers breaking flow, bullet spam, emoji spam) fragments attention, signals lack of confidence, creates visual noise. List addiction (5 Ways to X, 7 Secrets) flattens complexity, sacrifices depth for clickability. Redundant transitions ("As mentioned," "Moving on," "In conclusion," "To summarize") suggest unclear thinking—structure should show relationships.
end

PASS RhetoricalAntipatterns:
    sutra: "Language patterns that weaken claims or obscure mechanism through imprecision."
    gloss:
        Hedging excess ("I think perhaps maybe...") signals uncertainty masquerading as humility, weakens without precision. False precision (specific timelines without basis, numerical claims ungrounded, definitional authority without grounding) damages credibility. Anthropomorphizing ("Framework wants," "System feels," "Let practice guide") mystifies, creates false agency, obscures mechanism. Philosophical maximalism ("Cosmic emanations," "Species transformation," "Transcendent evolution") overpromises, alienates, replaces substance with grandiosity.
end

PASS EngagementAntipatterns:
    sutra: "Manipulative patterns prioritizing conversion or metrics over genuine value."
    gloss:
        Engagement-baiting ("Comment if agree!", "Like if resonated!", "Double-tap for more") signals desperation, manipulates connection, cheapens value. Manufactured urgency ("LIMITED TIME," "Don't miss," "NEED to understand NOW") creates false pressure, damages trust, reveals prioritization of conversion. Humble-bragging ("Just finished 12-hour session," "Grateful for amazing journey," "Blessed to share") performs authenticity rather than embodying it, signals status anxiety.
end

PASS ContentAntipatterns:
    sutra: "Content-level patterns violating accessibility or intellectual commitment."
    gloss:
        Jargon as gatekeeping uses technical terms to exclude, builds unnecessary complexity signaling sophistication, signals insecurity. False simplification ("Actually quite simple," "Just do X and Y happens," "The secret") dishonors complexity, creates unrealistic expectations, fails pedagogical responsibility. Ungrounded authority claims without logic, "trust me," "everyone knows"—demands belief rather than earning understanding.
end
