### 5.3 The Unified Folder Purpose

The `prompts/unified/` folder serves a specific architectural function:

**Canonical** (prompts/canonical/):
- Platform: ChatGPT, Claude API, Gemini, Grok
- Format: Two-field split (More About You + Custom Instructions)
- Constraint: Character limits per platform
- Use: Direct paste into platform settings

**Unified** (prompts/unified/):
- Platform: Claude Projects, Custom GPTs with extended context
- Format: Full XML-structured specification
- Constraint: None (can be comprehensive)
- Use: Project Files upload or Knowledge Base

**gemknowledge-base**:
- Platform: Google Gems
- Format: Structured knowledge base format
- Use: Gem configuration

This separation honors platform-specific constraints while maintaining a single source of truth in the synthesis files.
