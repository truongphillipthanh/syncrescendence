## 1. Executive Synthesis

The first swarm wave converged on a usable constitutional answer.

The live shell is strong enough to serve as the receiving institution, but not yet contractually complete enough to ingest semantic mass without relapse. The older inhabited shell remains the main authenticity and burden-bearing witness. The pre-schematic archive remains the richest epistemic treasury and redesign laboratory. None of the three can replace the others wholesale.

The governing adjudication is therefore:

- `current shell` = live receiver and only present authority surface
- `syncrescendence.old` = authenticity reserve and operational memory quarry
- `syncrescendence_pre_schematic_design` = semantic treasury and doctrine/rebuild reserve

The repo should become smaller in raw mass and stronger in ratified meaning.

That implies:

- migrate by authority class, not by folder nostalgia
- preserve pedigree as a first-class destination
- keep doctrine, the secondary knowledge substrate, operator, agent-local memory, and archive receipts distinct
- externalize large raw source mass once compacted or manifested
- keep exocortex as coordination field, ontology as projection field, and repo as ratification field

## 2. Stable Convergences

The workers and coordinator produced the following high-confidence consensus.

The current shell is viable as receiver.

- `communications/`

- the contract between repo authority, exocortex state, and ontology projection

### 2.2 Doctrinal verdict

The strongest live-law seeds come primarily from `neocanon`, with selective reinforcement from old canon where compression removed important burden-bearing texture.

The strongest lower-tier knowledge seeds come primarily from `neocorpus` and `neosyncrescendence`.

The old canon should not be restored as shell topology or total front-door authority. It should be mined as a reserve of authentic doctrine, especially where it preserves:

- memory sovereignty
- context transition
- research rigor
- artifact protocol
- implementation discipline
- feedcraft and platform sovereignty
- progressive disclosure

### 2.3 Operational verdict

Most of `05-SIGMA` is not constitutional scripture. Its yield is procedural and should be rehomed into:

- `playbooks/`
- `operators/`
- `validated-patterns/`
- `pedigree/` for originals and cautionary exempla

Likewise, old sovereign and agent-local surfaces should be reconstituted by burden class:

- executive meaning into `executive/`
- office-local practice into `offices/`
- inter-office lineage into `communications/`
- historical originals into `pedigree/`

### 2.4 Sources verdict

The live repo should stop being the primary residence of large raw source families.

What should remain live:

- compacted lessons
- lawful manifests and receipts
- lineage anchors
- current control-plane state and operational doctrine
- small, repeatedly consulted reference items

What should trend outward:

- bulky raw source libraries
- stale logs
- superseded research sediment

### 2.5 Exocortex and ontology verdict

The unifier should not be a second canon.

The compact split is:

- exocortex coordinates
- ontology projects
- repo ratifies

Any direction-changing artifact must still land in the repo.

## 3. Adjudicated Decisions

The first wave left several collisions unresolved. They are now decided as follows.

### 3.1 One lawful disposition registry

There must be one canonical migration registry, not several local ledgers.

Every migration candidate gets:

- source tributary
- original path
- artifact class
- authority score
- present relevance
- compaction yield
- provenance sensitivity
- chosen disposition
- destination lane
- justification
- receipt pointer

Allowed dispositions remain:

- `promote_live_law`
- `promote_reference`
- `promote_playbook`
- `promote_operator`
- `promote_validated_pattern`
- `retain_pedigree`
- `retain_archive_manifest_only`
- `externalize_to_exocortex`
- `cull_with_receipt`

### 3.2 Canon / Sigma / Reference stratification

Batch 01 treated `knowledge/references/` as the current placeholder for the secondary knowledge tier below canon and above feedstock.

User clarification changes the framing:

- canon is closer to the whole-shell `CLAUDE.md` / `AGENTS.md` for Syncrescendence
- Syncrescript is the intended compression layer, not the replacement for canon
- `Sigma` was historically intended to inhabit the rung below canon
- the present name `references/` may therefore be only a temporary shell convenience

The current adjudication is therefore provisional:

- `knowledge/canon/` should hold the durable, consult-on-every-pass operative scripture of the shell
- the current lower tier should be re-evaluated in Batch 02 as one of:
  - a true `sigma/` lane
  - a retained `references/` lane with Sigma as the semantic class it houses
  - another name that better captures living harness-memory substrate without flattening it into generic reference
- `knowledge/feedstock/` remains the lawful intake floor

What is already clear:

- raw research and bulky source matter are not canon
- per-harness local memory is not canon
- the lower-than-canon tier is real even if its final name is not yet settled
- Batch 02 must decide the naming and promotion contract explicitly instead of assuming `references/` is correct

### 3.3 Pedigree is not a dumping ground

`pedigree/` must distinguish:

- preserved originals
- archive manifests
- rehousing receipts
- cautionary artifacts
- historical executive/state lineage

Anything retained for ancestry without present authority should be legible there, not hidden in live lanes.

### 3.4 Agents / offices, executive, and communications

The burden split is:

- `offices/` in the current shell, potentially `agents/` in the future = local working state, bounded queues, journals, scratch, local memory
- `communications/` = inter-office prompts, dispatches, receipts, results, confirms, returned lineage
- `executive/` = direction-changing narratives, escalations, summit outcomes, live steering surfaces

An artifact does not become executive merely because it is important. It becomes executive when it changes sovereign steering, priority, or legitimacy.

What remains open after user clarification:

- whether the physical directory should revert from `offices/` to `agents/`
- whether agent-local directories should be named by harness to reduce schizophrenia
- whether avatarization should remain conceptual/discursive rather than physical-directory primary

Batch 02 must therefore treat physical naming as an explicit design question, not an already-settled one.

### 3.5 Duplicate doctrine merge rule

Where both `old` and `neo` witness the same doctrine family:

- `neo` wins for current shell fit and compression
- `old` wins where it preserves burden-bearing rationale or operational grain lost in compression
- merged live artifacts should cite both when both materially matter
- unresolved collisions should be recorded, not papered over by synthesis tone

The shell must prefer explicit dual witness over false singularity.

### 3.6 Source and log shedding

Raw matter stays live only if it still earns its cost.

For bulky sources and logs, the default path is:

1. compact if net-positive insight remains
2. manifest if provenance matters but body does not need to stay live
3. cull with receipt if neither authority nor epistemic yield survives

### 3.7 Repo / exocortex / ontology non-duplication

The repo is the authority surface.

The exocortex may hold live coordination state.

The ontology may project normalized structure and linkage.

Neither exocortex nor ontology may silently become the semantic authority for doctrine, legitimacy, or executive direction.

## 4. Holistic Evaluation

The swarm result is strong.

It did not simply rediscover folder mappings. It recovered the real structure of the migration problem:

- the bridge is not archival restoration
- the bridge is not shell beautification
- the bridge is a controlled rebinding of authority, memory, and scale

What must survive first are the laws for memory, context, artifact evolution, research discipline, office burden, and compaction. The older metaphysical and narrative material still matters, but as pedigree reserve unless and until it proves present governing utility.

The main remaining risks are:

1. canon stratification remains semantically confused because the Sigma tier is missing or misnamed
2. `pedigree/` stays under-specified and becomes a burial mound
3. agent-local doctrine quietly becomes hidden law
4. source shedding happens ad hoc without manifest discipline
5. ontology/exocortex convenience outruns repo ratification
6. duplicate doctrine is merged by stylistic preference instead of witness discipline
