## Extracted Patterns & Wisdom

### Decision Framework
- **18 Evaluative Lenses**: Every strategic decision scored; threshold 12/18 pass
- **Temporal content automatically fails**: Point-in-time snapshots fail Bitter Lesson Scaling, Antifragile, Permaculture

### Organizational Patterns
- **Flat Hierarchy + Metadata + Aliases**: Filesystem flat; naming encodes hierarchy; YAML metadata; Obsidian aliases
- **SCREAMING_SNAKE Convention**: Prefixes encode type; hyphens separate segments; underscores within
- **Cascade Principle**: Cosmos tier determines everything below; audit TOP-DOWN

### Content Processing
- **4-Tier Signal Classification**: paradigm / strategic / tactical / noise
- **Processing Pipeline**: SOURCE → triage → process → integrate → update records
- **"Every token fights for its place"**: Compression mandatory

### Execution Patterns
- **Parallel Streams**: Alpha + Beta execute simultaneously with clear handoff
- **Phased Execution**: Gate checks between phases; verification before proceeding
- **Blitzkrieg Model**: Lane A/B/C; parallel execution; model selection per task
- **Pre-execution backup**: Git tags provide rollback capability

### Critical Learnings
- **"Metabolism applies to CONTENT, not ORCHESTRATION"** — State files and logs are living infrastructure
- **"Don't lobotomize platforms to get determinism"** — Account for unreliability to get genuine intelligence
- **"Rivers flow into wells before evaporation"** — Conversations ephemeral; repository persistent
- **"The bottleneck is Sovereign attention, not information access"** — Infrastructure serves attention allocation
- **Category Error**: Never apply content deletion logic to operational infrastructure
