### Pattern 4: Structure != Substance
- **Oracle 4**: Achieved 79% reduction (substance)

- **Rule**: Focus on the actual work, not organizing the work.

### Pattern 5: Distillation Semantics

NOT: Create subdirectory hierarchy and move files.
