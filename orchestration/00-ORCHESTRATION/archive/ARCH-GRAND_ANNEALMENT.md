## 8. Seared Lessons Register

1. **launchd never sources ~/.zshrc — ever.** Env vars for launchd must be in plist
   `EnvironmentVariables`. `~` does not expand in plist XML — use absolute paths.
2. **`sudo` in recovery scripts registers plists in root domain, not user domain.** Fix:
   `launchctl bootstrap gui/$(id -u)`. First unplug test was 6-minute total failure from this.
3. **`grep config-file` proves config exists, not that it works.** Runtime verification =
   `ps eww -p $PID | grep VAR`. Always verify from within the actual execution context.
4. **Async verification gap destroys trust.** Fix → commit → dispatch audit → don't wait →
   commit next fix → audit returns FAIL → too late. NEVER commit without blocking verification.
5. **`/tmp` is cleared on macOS reboot.** PID lockfiles must live in `~/Library/Application
   Support/` for persistence across cold boots.
6. **`flock` is Linux-only; does not exist on macOS.** Use PID-file lock pattern.
7. **`git add -A` with no scope guard is a deletion bomb.** Files absent on one machine staged
   as deletions. git_sync.sh must scope to -INBOX/ and -OUTGOING/ only, abort if >20 deletions.
8. **Google Drive Desktop Backup + git repos = silent corruption.** Ghost branches, .DS_Store in
   .git/, duplicate index files. Apply xattr `com.apple.fileprovider.ignore=1` and
   `com.google.drivefs.ignore=1`. Permanent fix: disable Desktop backup in GDrive (GUI, pending).
9. **Dual watcher race swallows tasks silently.** watch_dispatch + auto_ingest both claim the
   same task. watch_dispatch also produces 0-byte output and hangs. NEVER re-enable it.
10. **ChatGPT Plus quota is a shared pool between Psyche and Adjudicator.** Saturating both
    simultaneously kills one. Adjudicator was dead 3 days, reporting Exit-Code: 0 throughout.
11. **Gemini TUI structurally rejects tmux send-keys.** ncurses input buffer ignores programmatic
    injection. Headless mode (`gemini -p -y -o text`) is the only valid orchestration path.
12. **Kimi K2.5 hallucinated task completion.** Wrote 18-line stub, claimed full delivery.
    Ajna/Kimi K2.5 is unreliable for file-writing analytical tasks; suitable for conversational
    and strategic advisory work only.
13. **Context decay kills 86% of deferred commitments.** 12/15 OPEN, 0 IN_PROGRESS after
    compaction. Structural mitigation: DYN-DEFERRED_COMMITMENTS.md + CLAUDE.md step 1b.
14. **Recovery chain has strict dependency ordering.** Docker before cockpit; tmux before
    auto-ingest. Race conditions produce partial recovery that appears like success.
15. **`git add -A` at commit time sweeps in-flight state.** Commit b64bd3c grabbed 36 extra
    in-flight files. Agents must use `git add <specific files>` — never `git add -A`.
16. **Self-confidence vs independent audit gap is 27 points.** Commander self-assessed 88%
    pipeline soundness. Adjudicator adversarial audit returned 61/100. Validation debt is real.
17. **ICMP ping is blocked by macOS Stealth Mode.** NEVER use ping for health checks.
    SSH health check: `ssh -o ConnectTimeout=5 mini hostname`.
18. **NEVER dispatch without Reply-To and CC fields.** One-way dispatch = coordination failure.

19. **27% of ClawHub's 5,705 skills are spam/malicious.** skill-vetting gate is mandatory before

20. **CARDINAL RULE: Never ask Sovereign to do what you can dispatch.** Violated 3 times before
