## QUICK CONTEXT CLUES

### Clues It's **Scale** (Cosmological Sophistication):
- "How much cosmology"
- "Micro/Meso/Macro/Meta"
- "Seven Pulses vs. Planetary"
- "Optional advancement"
- "Practitioner chooses complexity level"

### Clues It's **Level** (Chain Competence):
- Mentions specific chain (Technology, Sensing, etc.)
- "Foundation → Intermediate → Advanced → Mastery"
- "Requires previous level mastery"
- "4-level progression"
- "Chain-specific development"

### Clues It's **Degree** (Lunar Mastery):
- **5 levels** (not 4!)
- "Coherence" or "Transcendence" context
- "Recognition → Exploration → Commitment → Integration → Transmission"
- "Lunar system"
- "Epistemological" or "ontological" depth

### Clues It's **Stage** (Platform Maturity):
- Platform names: Newsletter, Twitter, Medium, YouTube, Book
- "Theatrical" names: Forum, Podium, Amphitheatre, Atrium, Portico
- "Content production"
- "Sequential organizational gating"
- "Platform capabilities"

### Clues It's **Phase** (Business Timeline):
- "Foundation/Validation/Scaling/Institute/Infrastructure"
- "Business operations"
- "Organizational maturity"
- "Chronological progression"
- NOT "phase of practice" (that's Level/Degree)

---

## COMMON TRAPS TO AVOID

### TRAP 1: "Tier" = Always the Same Thing
❌ **WRONG**: Replace all "Tier" with "Scale"  
✓ **RIGHT**: Context determines: Scale/Level/Stage/Degree/Phase

### TRAP 2: "Level" = Always Chain Progression
❌ **WRONG**: All "Level" → "Level: [1-4]"  
✓ **RIGHT**: Could be Degree (5-level Lunar), check context

### TRAP 3: Five Levels = Level
❌ **WRONG**: "Five levels of practice" = Level: [1-5]  
✓ **RIGHT**: 5 levels = ALWAYS Degree: [1-5] (Lunar only)

### TRAP 4: "Mastery" = Level: 4-Mastery
❌ **WRONG**: All "mastery" → "Level: 4-Mastery"  
✓ **RIGHT**: Could be Degree: 5-Transmission, Scale: 4-Meta, or Level: 4-Mastery

### TRAP 5: "Stage" = Always Platform
❌ **WRONG**: Keep all "Stage" unchanged  
✓ **RIGHT**: If chain context, replace with Level: [1-4]

## VALIDATION QUICK-CHECKS

After any replacement, verify:

1. **Does it have the required format?**
   - Scale: [1-4]-[Name] ✓
   - Level: [1-4]-[Name] [Chain] ✓ (chain required!)
   - Degree: [1-5]-[Name] ✓
   - Stage: [1-5]-[Name] ✓ (theatrical name!)
   - Phase: [Name] Phase ✓

2. **Is the context clear to a new reader?**
   - Would someone unfamiliar with the system understand?
   - Are all references unambiguous?

3. **Does it maintain orthogonality?**
   - No statements suggesting dimensional hierarchy
   - No impossible constraints (e.g., "Must reach Scale 3 before Level 2")

4. **Is it consistent with previous uses?**
   - Same concept = same terminology throughout document
