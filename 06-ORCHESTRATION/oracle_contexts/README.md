# Oracle Contexts

Strategic synthesis documents for each Oracle session.

## Naming Convention

```
ORACLE[NN]_CONTEXT_v[M].md     # Working context (versioned during session)
ORACLE[NN]_FINAL_STATE.md       # Terminal state (session complete)
ORACLE[NN+1]_HANDOFF.md         # Transition to next Oracle
```
