# Rehoused Pedigree Artifacts

This lane holds archived artifacts that have been copied back into the canonical shell under explicit provenance control.

Rules:
- the copied artifact remains byte-for-byte source-equivalent unless a later compaction pass explicitly derives a successor
- every rehoused artifact must have a matching receipt in [rehousing-receipts](/Users/system/syncrescendence/pedigree/rehousing-receipts)
- rehousing is not promotion; it is lawful presence inside the shell for later interpretation, compaction, or playbook extraction
