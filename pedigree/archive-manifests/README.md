# Archive Shell Manifests

This lane holds comparative manifests for predecessor shells and archive snapshots.

Purpose:
- classify what each predecessor shell contains at the top level
- assign provisional re-ingestion dispositions
- distinguish live-shell candidates from pedigree-only material
- make lossless migration tractable instead of intuitive

These manifests are not promotion decisions by themselves.
They are the first-pass intake maps used to drive later tranche work.
