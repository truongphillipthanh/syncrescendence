## The Three Axioms (Core Principles)

### Axiom 1: Displacement Mechanism
Intelligence displaces tools by **collapsing the means-ends distinction**. 

Traditional tools require human intelligence to select tool, sequence operations, handle edge cases, integrate results. AI collapses this: you specify *what*, intelligence determines *how*.

This isn't augmentation—it's **categorical displacement** of the coordination function.

### Axiom 2: Intent Architecture Primacy
Human cognitive work decomposes into hierarchical intent:
```
Vision → Mission → Strategy → Projects → Processes → Tasks → Operations
```

The Personal Ontology formalizes this hierarchy, with intelligence mediating every level.

### Axiom 3: Phase Transition Law
Adoption follows predictable developmental stages:
```
Pre-Integration → Copiloting → Delegation → Symbiosis → Autonomy
```

Each phase requires different cognitive architecture and interaction patterns. Frameworks must be **generative**—describing not just current state but evolutionary trajectory.

## The Three Modes (Invariant Work Patterns)

### Mode 1: CREATION (Potential → Actual)
**Essence**: Bringing new artifacts into existence

**Human role**: Vision, taste, approval
**AI role**: Execution, iteration, exploration

**Pattern**: Specify → Generate → Select → Refine

### Mode 2: TRANSFORMATION (Actual → Refined)
**Essence**: Improving existing artifacts

**Human role**: Judgment about "better", approval gates
**AI role**: Analysis, optimization, variant generation

**Pattern**: Provide + Intent → Review Changes → Iterate

### Mode 3: COMPREHENSION (Complexity → Clarity)
**Essence**: Understanding information

**Human role**: Right questions, integration into worldview
**AI role**: Research, analysis, synthesis, explanation

**Pattern**: Frame Question → AI Researches → Dive Deeper → Reach Conclusion
