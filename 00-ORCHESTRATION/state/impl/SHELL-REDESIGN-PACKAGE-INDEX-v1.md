## Current Status

This package is design law, not yet migration execution.

It does **not** yet:

- move live files,
- create all successor lanes,
- enforce validators,
- reconstitute the live Rosetta or Intent Compass,
- or compact playbook material into finished playbooks.

It **does** define the legal shell those future operations must obey.

The first executable derivative now exists:

- inventory-mode artifact-law scan via `make artifact-law-inventory`
