### PROC::QualificationDecision
```
sutra: Four-question assessment for any content

steps:
  1: abstraction >> does_level_match_current_learning?
  2: modality >> optimal_format? find_text/audio/code?
  3: cadence >> consuming_at_this_frequency_serves_objectives?
  4: fidelity >> source_trustworthy_enough_for_attention?

example:
  encountering: "Daily AI News Update" YouTube channel
  assessment:
    abstraction: L2-L3 (tactical → domain)
    modality: Visual-Dynamic
    cadence: Daily
    fidelity: T3-T4 (aggregator, not synthesizer)

  decision_acumen: PRUNE
  reason: "daily obligation + T3-4 fidelity + video inefficiency = better alternatives exist"

  decision_efficacy: CONDITIONAL_ARCHIVE
  reason: "if specific tutorials high-quality (T2), bookmark individuals. Don't subscribe daily."
```

### TERM::PathologyRecognition
```
sutra: Common curation failures + mitigation

spec:
  pathologies:
    treadmill_obligation:
      symptom: "constantly consuming to keep up"
      cause: too_many_daily_sources
      fix: signal_density > coverage_completeness | prune_ruthlessly

    abstraction_imbalance:
      symptom: "only L1-2 OR only L7-9"
      cause: unintentional_gradient_distribution
      fix: explicit_gradient_targets_per_chain

    modality_default:
      symptom: "only video" or "only text"
      cause: passive_format_selection
      fix: multi_modal_reinforcement | efficiency_optimization

    fidelity_confusion:
      symptom: "popularity = quality"
      cause: mistaking_reach_for_signal
      fix: explicit_tier_assessment | verify_credentials

    echo_chamber:
      symptom: "same perspective everywhere"
      cause: algorithmic_amplification
      fix: intentional_diverse_perspectives | steelman_opposing

    backlog_guilt:
      symptom: "47 unread, should read, never do"
      cause: subscribing_faster_than_consuming
      fix: AI_prescreening | ruthless_unsubscribe | batch_processing

  verification_protocol:
    surprising_claims:
      1: check_source_tier (T1-2 >> investigate | T3-4 >> verify_against_primary)
      2: cross_reference (2+ independent T1-2)
      3: trace_to_origin (find_primary_research)
      4: evaluate_methodology (if_research)
      5: suspend_judgment (better_uncertain_than_confidently_wrong)
```
