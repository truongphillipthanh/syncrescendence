## IV. THE CLAIM ATOM PRIMITIVE (ChatGPT's Contribution)

This is the engineering abstraction that makes the architecture concrete:

```yaml
# CLAIM ATOM SCHEMA
claim_atom:
  id: "CLAIM-YYYYMMDD-HHMMSS-{hash}"
  
  # Provenance
  source_url: "https://..."
  source_type: "youtube|podcast|paper|x_thread|blog"
  timestamp: "HH:MM:SS"  # for video/audio
  speaker: "Name or Unknown"
  
  # Content
  claim: "Falsifiable assertion in one sentence"
  supporting_quote: "Verbatim quote that substantiates claim"
  context: "2-3 sentences of surrounding context"
  
  # Metadata
  novelty: 1-10  # How new is this relative to CANON?
  relevance: 1-10  # How aligned with Syncrescendence?
  confidence: 1-10  # How certain is the source?
  fidelity: "primary|synthesis|commentary|aggregation"
  
  # Connections
  topic_tags: ["agentic-systems", "scaling-laws"]
  contradicts: ["CLAIM-ID-if-any"]
  supports: ["CLAIM-ID-if-any"]
  iic_chain: "acumen|coherence|efficacy|mastery|transcendence"
  canon_refs: ["CANON-25010", "CANON-00012"]
  
  # Lifecycle
  status: "raw|qualified|integrated|superseded"
  integrated_into: "CANON-ID or null"
```

**Why This Matters**: 
- Claims are the atomic unit, not transcripts
- Everything else (heat maps, briefs, synthesis) computes from atoms
- Atoms can be traced back to source
- Atoms can be falsified against CANON
